# flask-auth-app

# venv 開発環境構築
```
python3 -m venv env
```
```
source env/bin/activate
```
```
pip install -r requirements.txt
```
```
export FLASK_APP=project
```
```
export FLASK_DEBUG=1
```
```
FLASK_ENV=development
```

docker compose up -d
